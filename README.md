# VAYUHARA ORIGIN | Premium Startup Website

A professional, modern, and clean digital creative agency website for **VAYUHARA ORIGIN**. Built with a deep navy and amber design system, featuring advanced parallax animations and high-performance scroll triggers.

## 🚀 Key Features

*   **💎 Premium Startup Aesthetic**: Clean, minimalist corporate look with a focus on high-quality visuals and typography.
*   **🌓 Smooth Interactivity**: Advanced parallax mouse movement on the hero logo and section-by-section scroll reveal.
*   **🏎️ Optimized Assets**: High-performance JPEG logo integration and font-weight optimization for fast loading times.
*   **📁 Custom Branded Content**: 
    - **Sticky Navbar**: Features the brand's unique JPEG logo prominently.
    - **Interactive Process Timeline**: Clearly displays the Discover, Design, Develop, and Launch roadmap.
*   **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
*   **📧 Contact System**: Professional, animated contact form with live submission feedback.

## 🛠️ Tech Stack

*   **Structure**: Semantic HTML5
*   **Styling**: Modern CSS (Vanilla) with custom Bezier transitions and design tokens.
*   **Functionality**: Vanilla JavaScript (ES6) for smooth scrolling and parallax engines.
*   **Icons**: Font Awesome (6.4.0)

## 📦 Getting Started

1.  Clone/Download the repository.
2.  Open `index.html` in any modern web browser to view the project.
3.  For live development, we recommend using a local server (like "Live Server" for VS Code).
